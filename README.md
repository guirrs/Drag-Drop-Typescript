## 📌 Sobre o Projeto
Este é um gerenciador de projetos dinâmico que permite criar, organizar e transitar tarefas entre estados (**Ativo/Concluído**) utilizando a funcionalidade nativa de **Drag and Drop**.

O sistema foi desenvolvido com fins estritamente **educacionais** para colocar em prática a lógica de **TypeScript**, explorando Decoradores e a Tipagem.

---

## 🚀 Tecnologias e Ferramentas

* **Linguagem:** TypeScript
* **Ambiente:** Node.js v24.13.1 (LTS)
* **Servidor Local:** Live Server (VS Code Extension)

---

## 🛠️ Instalação e Execução

### 1. Pré-requisitos (Node.js)
Para compilar e modificar o projeto, você precisa do Node.js instalado. Recomenda-se a versão **LTS** para maior estabilidade.

* 📥 **[Clique aqui para baixar o Node.js Atualizado (Site Oficial)](https://nodejs.org/en/download)**


### 2. Configuração do Ambiente
Abra o terminal na pasta do projeto e execute:

```bash
# Instalar o TypeScript como dependência de desenvolvimento

# Compilar o código TypeScript para JavaScript pelo webpack
